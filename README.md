# WARALA

**WARALA** is a mobile application developed to support **marine conservation, environmental awareness, and community engagement**. The application enables users to learn about marine ecosystems, report pollution incidents, participate in cleanup events, and contribute to protecting Sri Lanka’s coastal environment.

## Features

### Splash Screens
- Interactive onboarding experience
- Introduction to WARALA's mission and objectives
- User-friendly navigation between onboarding screens

### User Authentication
- User Registration (Sign Up)
- User Login
- Firebase Authentication Integration
- Password Reset Functionality

### Home Module
- View upcoming marine conservation events
- Explore environmental activities
- Quick access to reporting features

### Learn Module
- Marine species information
- Coral reef awareness
- Ocean ecosystem education
- Environmental facts and learning resources

### Pollution Reporting
- Report pollution incidents
- Upload relevant information
- Community contribution system

### Community Events
- Beach cleanup activities
- Volunteer participation
- Event details and registration

### Additional Pages
- About Us
- Contact Us
- FAQ Section
